# Система учёта посещаемости студентов

Веб-приложение на **Python + Reflex** для автоматизации учёта посещаемости: группы, студенты, дисциплины, занятия, отметки и статистика.

Интерфейс в сине-белых университетских цветах. Список группы можно загрузить из Excel.

## Требования

- Python 3.11+
- Visual Studio Code 
- Node.js (Reflex подтянет фронтенд автоматически при первом запуске)

## Установка в Visual Studio Code

1. Откройте папку проекта:
   - **File → Open Folder…** → выберите `student-attendance`
2. Откройте терминал: **Terminal → New Terminal**
3. Создайте и активируйте виртуальное окружение (если ещё не создано):

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

4. Установите зависимости:

```powershell
pip install -r requirements.txt
```

5. Инициализируйте базу данных (один раз):

```powershell
reflex db init
reflex db makemigrations --message "initial"
reflex db migrate
```

6. Запустите приложение:

```powershell
reflex run
```

7. Откройте в браузере адрес из терминала (обычно http://localhost:3000).

### Рекомендуемое расширение VS Code

- **Python** (Microsoft) — для работы с `.venv` и отладки.

В VS Code выберите интерпретатор: `Ctrl+Shift+P` → **Python: Select Interpreter** → `.venv\Scripts\python.exe`.

## Импорт студентов из Excel

1. Создайте группу (например `ПИ-21`) на странице **Группы**.
2. На странице **Студенты** выберите эту группу и загрузите файл `.xlsx`.
3. Формат файла (см. шаблон `assets/shablon_studentov.xlsx`):

| ФИО |
| Иванов Иван Иванович |

Можно без заголовков: колонка A — ФИО

## Основной сценарий работы

1. Создать **группы**.
2. Добавить студентов вручную или **импортировать Excel**.
3. Добавить **дисциплины** (название + преподаватель).
4. Создать **занятия** (дата + группа + дисциплина).
5. На странице **Посещаемость** отметить статусы.
6. Смотреть **Статистику** и диаграмму по группе.

## Статусы посещаемости

- Присутствовал
- Отсутствовал
- Уважительная причина
- Опоздал
